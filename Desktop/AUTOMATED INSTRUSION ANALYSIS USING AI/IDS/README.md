# IDS — Intrusion Detection System

An Explainable AI-powered Network Intrusion Detection System built with React, TypeScript, and Vite.

## Technologies Used

This project is built with:

- **Vite** — fast build tool and dev server
- **TypeScript** — typed JavaScript
- **React** — UI framework
- **shadcn-ui** — component library
- **Tailwind CSS** — utility-first styling

## Getting Started

The only requirement is having **Node.js & npm** installed — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Editing the Code

**Use your preferred IDE**

Clone the repo, make your changes locally, and push them back to the repository.

**Edit a file directly in GitHub**

- Navigate to the desired file.
- Click the **Edit** button (pencil icon) at the top right.
- Make your changes and commit.

**Use GitHub Codespaces**

- Go to the main page of your repository.
- Click the **Code** button → **Codespaces** tab → **New codespace**.
- Edit files directly and push your changes when done.

## Deployment

Build the production bundle with:

```sh
npm run build
```

Then deploy the contents of the `dist/` folder to any static hosting provider (e.g. Firebase Hosting, Netlify, Vercel).
